# 🛒 DecodeLab Shop (Project 1–3)

A full-stack e-commerce web application built as part of the DecodeLab training program.  
This project demonstrates frontend development, backend API creation, and database integration using MongoDB Atlas.

---

## 🚀 Project Overview

DecodeLab Shop is a simple online store where users can:

- View products from a database
- Add products to cart (UI counter)
- Edit products (admin feature)
- Delete products (admin feature)
- View services section
- Browse products dynamically loaded from backend API

---
